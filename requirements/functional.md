# Functional Requirements

## Customer:

1. As a customer, I want to browse meals offered by the catering service so that I can see what meal options are available.

2. As a customer, I want to view the list of ingredients required for each meal so that I can understand what ingredients I need to order.

3. As a customer, I want to order the ingredients for a selected meal so that I can receive everything needed to prepare it.

4. As a customer, I want to schedule ingredient deliveries (same-day or next-day) so that I can plan when to receive my ordered ingredients.

5. As a customer, I want to track the status of my orders so that I can know when my ingredients will be delivered.

6. As a customer, I want to search meals by name, ingredient, or category (e.g., vegan, quick meals, gluten-free) so that I can quickly find meals that match my preferences or dietary requirements.

## Admin/ Catering Manager:

7. As a catering manager / admin, I want to add, edit, and remove meals from the service so that the list of offered meals remains accurate and up to date.

8. As a catering manager/ admin , I want to update the status of orders (e.g., processing, shipped, delivered) so that customers can track their orders accurately.
